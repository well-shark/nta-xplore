https://transparencyreport.google.com/https/overview

- 所有 Google 产品和服务中的已加密流量 (Encrypted traffic across Google)
- 向 Google 提交的加密请求数（全球） (Encrypted requests to Google worldwide)
- Chrome 中的 HTTPS 加密情况（按平台） (HTTPS Encryption by Chrome platform)
  - Chrome 中通过 HTTPS 加载的网页所占的百分比（按平台） (Percentage of pages loaded over HTTPS in Chrome by platform)
  - Chrome 中的 HTTPS 浏览时间所占的百分比（按平台） (Percentage of HTTPS browsing time by Chrome platform)
- Chrome 中的 HTTPS 使用情况（全球） (HTTPS usage in Chrome worldwide)
  - Chrome 中通过 HTTPS 加载的网页所占的百分比（按国家/地区） (Percentage of pages loaded over HTTPS in Chrome by country/region)
  - Chrome 中的 HTTPS 浏览时间所占的百分比（按国家/地区） (Percentage of HTTPS browsing time in Chrome by country/region)
- 各大热门网站的 HTTPS 实施情况 (HTTPS on top sites)
  - 默认使用 HTTPS 的网站 (Sites that default to HTTPS)
  - 支持 HTTPS 的网站 (Sites that work on HTTPS)
